Element Selector (h2)

h2 { color: blue; }

This heading is blue.

This is not affected because of different hierarchy

ID Selector (#main-title)

#main-title { background: yellow; }

This heading has a yellow background.

Class Selector (.highlight)

.highlight { background: lightgreen; }

This paragraph has a green background.

Descendant Selector (section p)

section p { color: green; }

This paragraph inside a section is green.

Child Selector (section > p)

section > p { font-weight: bold; }

This direct child paragraph is bold.

This nested paragraph is NOT affected.

Adjacent Sibling Selector (h2 + p)

h2 + p { color: red; }

Heading

This paragraph right after h2 is red.

This paragraph is NOT affected.

General Sibling Selector (h2 ~ p)

h2 ~ p { text-decoration: underline; }

Heading

This paragraph after h2 is underlined.

This one is also underlined.

Attribute Selector (input[type="text"])

input[type="text"] { border: 2px solid blue; }
